The Program

How It Works

A full robotics experience, designed to drop into any enrichment or after-school program with zero friction.

01
Build the AC-1

Students assemble their robot from modular components, learning mechanical structure, servo placement, and how joints create movement.

02
Name It. Own It.

Each student personalizes their robot with a name. Ownership builds investment. Investment drives learning far beyond what any worksheet can.

03
Code It to Move

Using block-based programming, students write sequences that make their robot walk, wave, and respond. Every line of code has an immediate, visible result.
